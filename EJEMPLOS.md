# 💬 Ejemplos de Conversaciones

## 📊 Ejemplo 1: Consulta sobre Régimen Fiscal

**Usuario:**
```
Hola, tengo un negocio pequeño y gano como $50,000 al mes. ¿Qué régimen fiscal me conviene?
```

**ContaBot:**
```
¡Hola! 👋 Con ingresos de $50,000 mensuales ($600,000 anuales), tienes varias opciones:

**RESICO (Régimen Simplificado de Confianza)** ✅
- Tasa de ISR: 1.5% - 2.5% según tus ingresos
- Declaraciones bimestrales
- Menos obligaciones contables
- Ideal para tu nivel de ingresos

**Régimen de Actividades Empresariales**
- Tasa de ISR progresiva (hasta 35%)
- Más deducciones permitidas
- Mayor control contable

Para tu caso, el RESICO sería más conveniente por su simplicidad y menor carga fiscal. 

¿Te gustaría saber más sobre los requisitos del RESICO o tienes otra duda? 😊
```

## 🔄 Ejemplo 6: Conversación con Contexto

**Usuario:**
```
¿Qué es el IVA?
```

**ContaBot:**
```
¡Hola! 👋

El IVA (Impuesto al Valor Agregado) es un impuesto indirecto que se aplica al consumo de bienes y servicios en México.

**Características principales:**
📊 Tasa general: 16%
📊 Tasa fronteriza: 8%
📊 Tasa 0%: Alimentos básicos, medicinas
📊 Exentos: Servicios médicos, educación

**¿Cómo funciona?**
- Lo cobras a tus clientes (IVA trasladado)
- Lo pagas a tus proveedores (IVA acreditable)
- Declaras la diferencia mensualmente

¿Tienes alguna duda específica sobre el IVA? 😊
```

**Usuario:**
```
¿Y cuándo se declara?
```

**ContaBot:**
```
El IVA se declara **mensualmente** 📅

**Fecha límite:**
- Día 17 del mes siguiente al que declaras
- Ejemplo: IVA de enero se declara antes del 17 de febrero

**Importante:** ⚠️
- Aunque no tengas operaciones, debes presentar declaración en ceros
- Puedes hacerlo desde el portal del SAT
- Si no declaras a tiempo, hay multas y recargos

¿Necesitas ayuda con el proceso de declaración o tienes otra duda? 😊
```

## 🎯 Ejemplo 7: Pregunta Compleja

**Usuario:**
```
Soy freelancer, trabajo para empresas de USA y México. ¿Cómo declaro mis ingresos?
```

**ContaBot:**
```
¡Interesante caso! 🌎

Como freelancer con ingresos del extranjero:

**Régimen Fiscal:**
- Actividades Empresariales y Profesionales
- O RESICO (si calificas por monto de ingresos)

**Ingresos del extranjero:**
✅ SÍ debes declararlos en México
✅ Se consideran en pesos al tipo de cambio del día
✅ Puedes acreditar impuestos pagados en USA (evitar doble tributación)

**Facturación:**
- Clientes mexicanos: CFDI normal
- Clientes extranjeros: CFDI de exportación de servicios

**Importante:** ⚠️
Este es un caso que requiere análisis detallado de:
- Tratados fiscales México-USA
- Retenciones en origen
- Comprobantes de ingresos extranjeros

**Te recomiendo agendar una cita con un asesor** para revisar tu caso específico y optimizar tu situación fiscal. 👨‍💼

¿Quieres que un asesor se ponga en contacto contigo? 😊
```

## 💡 Tips para Personalizar

Puedes personalizar el comportamiento del bot editando `config.py`:

### Cambiar el tono de las respuestas
```python
SYSTEM_PROMPT = """Eres ContaBot, un asistente MUY formal y técnico..."""
```

### Agregar más palabras clave para asesor humano
```python
HUMAN_ADVISOR_KEYWORDS = [
    'asesor',
    'cita',
    'urgente',
    'problema grave',
    'demanda',
    'auditoría'
]
```

### Personalizar mensaje de derivación
```python
HUMAN_ADVISOR_MESSAGE = """
¡Claro! Un experto te contactará pronto.

📞 Llámanos: 55-1234-5678
📧 Email: contacto@tuempresa.com
🌐 Web: www.tuempresa.com
"""
```

## 🧪 Probar Localmente

Puedes probar el bot sin WhatsApp usando curl:

```bash
# Simular mensaje entrante
curl -X POST http://localhost:5000/webhook \
  -d "From=whatsapp:+5215512345678" \
  -d "Body=Hola, ¿qué es el ISR?"

# Ver estadísticas
curl http://localhost:5000/stats

# Limpiar conversación
curl -X POST http://localhost:5000/clear/whatsapp:+5215512345678
```

## 📚 Recursos Adicionales

- **Personalizar system prompt**: Edita `SYSTEM_PROMPT` en `config.py`
- **Cambiar límite de historial**: Modifica `MAX_HISTORY_MESSAGES` en `config.py`
- **Ajustar timeout de conversación**: Cambia `CONVERSATION_TIMEOUT` en `config.py`
